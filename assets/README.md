# Assets

This directory holds static media for the project — the logo, screenshots, and
terminal recordings referenced from the main [`README.md`](../README.md).

## Expected files

| File              | Purpose                                                        |
| ----------------- | ------------------------------------------------------------- |
| `logo.svg`        | Project logo (vector). Used at the top of the main README.    |
| `demo.gif`        | Terminal cast / animated demo of `devops-ai` analyzing input. |
| `architecture.png`| Diagram of the shared `AnalysisEngine` and the CLI/SDK/REST surfaces. |

These filenames are placeholders referenced by the main README; drop the real
assets in with these exact names so the links resolve. Keep large binaries
reasonable in size (the `check-added-large-files` pre-commit hook will flag
oversized files).
